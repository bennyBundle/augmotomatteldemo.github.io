Question: I can't connect my smart device to the track set/bluetooth isn't working

If after restarting the app your smart device is still not connecting to the track set, try these different steps:

  1. Power cycle track set - Soft reset
    1. Turn the track set OFF
    2. Wait 5-10 seconds.
    3. Turn the track set ON.
    4. Bluetooth indicator (blue) LED on track set should be blinking after power cycle. If it is not, proceed to next solution.
  2. If track set Bluetooth indicator is illuminated solid blue (not flashing) while attempting to connect
    1. Turn the track set OFF
    2. Wait 5-10 seconds.
    3. Turn the track set ON.
    4. Bluetooth indicator (blue) LED on track set should be blinking after power cycle. If it is not, proceed to next solution.
